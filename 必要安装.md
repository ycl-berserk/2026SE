## 一、必须安装的软件

### 1. Java 开发工具包（JDK 17）

**为什么需要**：后端使用 Spring Boot 3.x，必须依赖 Java 17 或更高版本。

#### 安装步骤（Windows）

**方式一：推荐 - 使用安装包**

1. 下载 JDK 17：
   - 官方地址：https://www.oracle.com/java/technologies/javase/jdk17-archive-downloads.html
   - 或国内镜像：https://mirrors.tuna.tsinghua.edu.cn/Adoptium/17/jdk/x64/windows/
   - 选择 `jdk-17.x.x-hotspot.msi` 安装包

2. 双击安装包，一路 Next 完成安装

3. 验证安装：
   ```bash
   # 打开命令提示符（Win + R，输入 cmd，回车）
   java -version
   ```
   应显示类似：
   ```
   java version "17.0.x"
   ```

**方式二：使用包管理器（适合有经验的同学）**

```bash
# 使用 winget（Windows 10/11 自带）
winget install Microsoft.OpenJDK.17

# 或使用 scoop
scoop install java/openjdk
```

#### 环境变量配置（如果安装后 `java -version` 不生效）

1. 右键"此电脑" → "属性" → "高级系统设置" → "环境变量"
2. 在"系统变量"中找到 `Path`，点击"编辑"
3. 添加 JDK 的 `bin` 目录路径（例如：`C:\Program Files\Java\jdk-17\bin`）
4. 点击确定保存，**重新打开**命令提示符验证

### 2. Git 版本控制工具

**为什么需要**：用于拉取项目代码、提交自己的修改。

#### 安装步骤

1. 下载 Git：https://git-scm.com/download/win
2. 双击安装包，保持默认选项，一路 Next
3. 验证安装：
   ```bash

   ```

#### 首次配置

```bash

git config --global user.name "你的姓名"
git config --global user.email "你的邮箱"

# 查看配置
git config --list
```

### 4. 人大金仓数据库（KingbaseES）

**为什么需要**：后端存储所有业务数据。

#### 安装步骤

> ⚠️ **注意**：如果团队已有成员安装并配置好数据库，其他人可以**远程连接**该数据库，无需重复安装。

1. 从课程提供的渠道获取安装包（通常在课程群或网盘中）
2. 运行安装程序，选择"典型安装"
3. 安装过程中会要求设置：
   - **端口号**：默认 `54321`（不要修改）
   - **超级管理员密码**：设置为 `123456`（或团队约定密码）
   - **字符集**：选择 `UTF8`

4. 安装完成后，验证服务是否运行：
   - 打开"服务"（Win + R，输入 `services.msc`）
   - 查找 `KingbaseES` 相关服务，确保状态为"正在运行"

## 四、数据库配置

### 1. 确认数据库连接信息

后端配置文件位置：`demo/src/main/resources/application.properties`

当前配置（**团队统一配置**）：

```properties
spring.datasource.driver-class-name=org.postgresql.Driver
spring.datasource.url=jdbc:postgresql://localhost:54321/test
spring.datasource.username=system
spring.datasource.password=123456
```

### 2. 如果数据库不在本机

需要修改上述配置中的：
- `localhost` → 改为数据库服务器的 IP 地址
- `password` → 改为实际密码

### 3. 数据库迁移（Flyway）

> ✅ **好消息**：项目启动时会自动执行数据库迁移，**不需要手动操作**！

首次启动后端时，Flyway 会自动：
- 创建 23 张业务表
- 插入初始基础数据（角色、分类、测试用户等）
- 记录迁移历史

## 五、启动后端服务

### 1. 打开命令提示符

```bash

```



## 七、常见问题排查

### ❌ 问题 1：`java -version` 提示"不是内部或外部命令"

**原因**：环境变量未配置或配置错误

**解决**：
1. 确认 JDK 安装路径（通常在 `C:\Program Files\Java\jdk-17\`）
2. 按照本文"环境变量配置"部分重新配置
3. **必须重新打开**命令提示符才能生效

---

### ❌ 问题 2：后端启动报错 `Connection refused`

**原因**：数据库未启动或连接信息错误

**解决**：
1. 确认人大金仓数据库服务正在运行（查看"服务"管理器）
2. 检查 `application.properties` 中的数据库地址、端口、密码
3. 尝试 ping 数据库服务器：`ping localhost`

---

### ❌ 问题 3：Flyway 迁移失败

**可能原因**：
- 数据库权限不足
- 已有表结构与迁移脚本冲突

**解决**：
```bash
# 清空数据库重新迁移（仅限开发环境！）
# 使用项目自带的清理工具
cd D:\2026SE\demo
mvnw.cmd compile exec:java -Dexec.mainClass="com.example.demo.DatabaseCleaner"
```

---

### ❌ 问题 4：微信开发者工具编译报错

**可能原因**：
- 项目目录选择错误（应该选择 `miniprogram-1` 而不是 `2026SE`）
- `project.config.json` 配置错误

**解决**：
1. 确认导入的目录是 `miniprogram-1`
2. 复制 `project.config.template.json` 为 `project.config.json`
3. 修改其中的 `appid` 为你的真实 AppID

---

### ❌ 问题 5：端口 8080 被占用

**解决**：
```bash
# 查看占用 8080 端口的进程
netstat -ano | findstr :8080

# 结束该进程（替换 PID 为实际数字）
taskkill /F /PID <PID>
```

或者修改后端端口：编辑 `application.properties`，添加：
```properties
server.port=8081
```
