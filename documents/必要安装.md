# 团队成员必要安装说明（当前版本）

> 适用对象：本项目所有成员  
> 更新时间：2026-04-22  
> 当前架构：学生端小程序 + 辅导员/管理员 Vue 网页端 + Spring Boot 后端

## 1. 必装软件

### 1.1 JDK 17

- 后端基于 Spring Boot 3.x，必须使用 JDK 17+
- 验证命令：

```bash
java -version
```

### 1.2 Node.js 20+

- `web-counselor` 需要 Node 环境
- 验证命令：

```bash
node -v
npm -v
```

### 1.3 Git

- 用于拉取代码和提交
- 验证命令：

```bash
git --version
```

### 1.4 微信开发者工具

- 用于运行 `miniprogram-1`
- 下载地址：[https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)

## 3. 本地启动（推荐顺序）

### 3.1 启动后端

```bash
cd demo
mvnw.cmd spring-boot:run
```

- 默认端口：`18080`
- 启动后会自动执行 Flyway 迁移并初始化演示数据（含请假场景）

### 3.2 启动网页审批端

```bash
cd web-counselor
npm install
npm run dev
```

- 默认地址：`http://localhost:5173`
- 首次可访问 `/login`
- 前端默认请求后端：`http://127.0.0.1:18080`

### 3.3 启动学生端小程序

1. 微信开发者工具导入 `miniprogram-1/`
2. 关闭域名校验（开发环境）
3. 检查 `miniprogram-1/utils/config.js` 的 `BASE_URL` 指向后端

## 4. 常见问题

### 4.1 后端端口冲突

```bash
cd demo
mvnw.cmd spring-boot:run "-Dspring-boot.run.arguments=--server.port=18081"
```

### 4.2 网页端请求不到后端

- 确认后端是否运行在 `18080`
- 如后端改端口，请同步修改 `web-counselor/src/api/http.js` 的 `baseURL`

### 4.3 小程序下载卡住或请求失败

- 开发者工具里关闭域名校验
- 检查本机防火墙和代理设置
- 确认 `BASE_URL` 使用可访问的局域网或本机地址
